# Concepts

This section of the docs covers all major aspects of LaunchMenu and explains how they work an interact with one and another.

These docs are divided in the following parts:

-   [Applet format](/docs/concepts/applet-format): Covers the format of the data applets can export
-   base: Covers core structural aspects of LaunchMenu
-   UI: Covers user interface elements of LaunchMenu, such as menus and textfields
-   Interaction: Covers components used for user interaction with elements
-   [Settings](/docs/concepts/settings): Covers how the settings system works
